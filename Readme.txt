Aplikacja Pinboard została wykonana w środowisku chmurowym Cloud9.

Uruchomienie aplikacju w środowisku lokalnym:

- preferowany system opracyjny: Linux lub Mac OS;
- rails: 4.2.5
- ruby: 2.3.0p0 (2015-12-25 revision 53290) [x86_64-linux]
- baza danych: 
    - środowisko programistyczne: Sqlite3
    - środowisko produkcyjne: PostgreSQL 
- server: WEBrick 1.3.1

Do poprawnego działania aplikacji konieczna instalacja ImageMagick.
